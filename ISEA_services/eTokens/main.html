For Department Administators

This section provides instructions and screen shots describing the typical functions that a department administrator may expect to encounter. In order to perform these tasks, the department administrator (DA for short), must have the following prerequisites:

  1. an eToken issued to themselves by the ITS eToken administrator.
  2. the desktop computer used to run the SafeNet Authentication Manager (SAM) must run a current Windows operating system.

The Internet Explorer web browser must be used to interact with SAM and must be configured as described in the Technical Information section.

You must have a supply of blank eTokens which can be used for issuing. Contact the ITS eToken administrator to obtain blank eTokens.

Enrolling a New User
  1. Insert a blank eToken.
  2. Access SAM by inserting your eToken, open Internet Explorer, and access the URL:
    3. https://ekey.utoronto.ca/sammanage/
  3. You will notice your UTORid in the upper right
  4. Select 'Deployment'
  5. 'Search for:' users by username
  6. 'Search criteria:' Enter the UTORid of the enrollee.
  7. Select 'Go'
  8. SAM will display the UTORid under the 'account name'. Ensure that the status of the account is: 'No token'
  9. Check the box beside the correct UTORid, select 'Enroll' button.
  10. SAM will process and stop at the following point:
  11. Select 'Run'. SAM will enroll the user and populate the eToken with a certificate. This will take up to a minute.
  12. Click the 'Done' button. The eToken is now ready for the user.
Notes: The DA cannot and should not assign an eToken to a person who already has one

Common Use Cases

The Cisco AnyConnect Client must be installed before using UTORvpn on windows.

  1. Download the Windows Client file from https://isops.noc.utoronto.ca/download/. You must have a valid UTORid and be eligible for the service.
  2. Run the installer file, and accept the licence agreement.
  3. You will need to be an administrator on your computer to finish installing the client. If you are on a managed workstation, contact your IT support to have them deploy the AnyConnect client.

The Cisco AnyConnect Client must be installed before using UTORvpn on OS X.

  1. Download the Mac Client file from https://isops.noc.utoronto.ca/download/. You must have a valid UTORid and be eligible for the service.
  2. Open the .dmg file, an administrator on your computer to do this.
  3. Double click on AnyConnect.pkg to start the installer.
  4. Click Continue and accept the Licence.
  5. Uncheck all modules except VPN. Click Continue.
  6. Complete the install

The Cisco AnyConnect Client must be installed before using UTORvpn on Linux.

  1. Download the Linux Client file from https://isops.noc.utoronto.ca/download/. You must have a valid UTORid and be eligible for the service.
  2. Navigate to your Downloads folder an extract tar.gz file to a directory.
  3. Navigate to the VPN directory in the extracted directory. Right click in the directory and choose Open In Terminal.
  4. Run the command: sudo bash vpn_install.sh and enter your password
  5. Agree to the licence agreement and the install will complete
