Weblogin Code Change

Written: August 2019

On Thursday August 29th, we will be introducing some minor code changes to the weblogin system used for web Single Sign On.

This change will allow external users from a specific institution to login in Quercus using their home institution accounts and credentials as part of a joint program.

This change will be transparent and should have no impact on our users.
