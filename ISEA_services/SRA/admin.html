AdminVPN is a service to provide strong authentication and secure network connections for Systems and Units at the University of Toronto.

For Systems (AdminsysVPN)

As a user, you must have an eToken, and the eToken client, for strong authentication. As well as a compatible VPN client.

Download the AnyConnect client from https://secure.sis.utoronto.ca/. The following installations are packaged for the following operating systems:
  • Windows: .msi
  • OS X: .dmg
  • Linux: .tar
hen installing from the VPN server, connect to https://port.eis.utoronto.ca with the eToken connected.

Usage

  1. Connect the eToken
  2. Start the AnyConnect client using either Firefox or Chrome. Enter the following URL: https://port.eis.utoronto.ca. The AnyConnect client will display a ‘Connected’ status on completion of connection. Note that connecting from the AnyConnect client does not work – a browser must be used.
  3. To disconnect the tunnel, use the ‘Disconnect’ button on the client.
